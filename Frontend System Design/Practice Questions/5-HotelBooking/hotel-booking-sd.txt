FUNCTIONAL REQUIREMENTS:
1) Search hotels: 
- Date range 
- Location
- Rooms
- No. of person
(on submit)

2) Display large result set with infinite scroll, pagination
3) Have additional filters - price range, ratings, amenities (on click)
3) Clicking opens hotel details page with photos, rooms and policies
4) Booking flow: Select room -> Guest info -> Payment 
5) Login, sign up, profile
6) Internationalization


NON-FUNCTIONAL REQUIREMENTS:
1) Accessibility 
2) Responsive 
3) Internationalization 
4) SEO support 
5) Testing 

ARCHITECTURE:
1) Rendering - SSR, ISR
2) State management - react-query, redux-toolkit
3) Styling - design system (Material UI, ShadCN), Tailwind CSS 
4) Bundling (Webpack)
5) Image optimization (CDN, avif/webp)
6) Authentication & authorization
